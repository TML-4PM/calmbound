# CalmBound

**Humane screen time boundaries for Australian families.**

## Quick Start - Kids Visit Mode (Free)

1. Download `packages/kvm/portal/index.html`
2. Upload to your router's captive portal
3. Configure guest network
4. Done!

## Structure

- `packages/kvm/` - Kids Visit Mode (open source)
- `packages/core/` - FastAPI backend
- `packages/evidence-plane/` - Measurement framework
